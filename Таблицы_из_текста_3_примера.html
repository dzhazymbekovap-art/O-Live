1
Прайс конкурентов → таблица сравнения рационов
Источник: скриншот с сайта конкурента (меню рационов)
📋 Промт
На скриншоте — меню рационов конкурента. Составь таблицу по параметрам: — Название рациона — Описание / цель рациона — Цена от (₸ в день) — Для кого подходит (определи сам по описанию) Отсортируй по цене от дешёвого к дорогому.

3
Условия поставщика → структурированная таблица требований
Источник: текст с условиями поставщика орехов (документация, финансы, логистика)
📋 Промт
Вот текст с условиями поставщика орехов. Составь таблицу: — Блок условий (документация / финансы / логистика) — Параметр — Конкретное требование / значение — Важность для нас (критично / важно / информация) Без лишних слов, только суть.
Блок Параметр Требование / значение Важность
Документация Декларация соответствия Обязательно по ТР ТС 021/2011 и 022/2011 Критично
Документация Качество сырья Без сора, плесени, вредителей, прогорклого запаха. Естественный цвет и влажность ядра Критично
Документация Протоколы испытаний Поставщик предоставляет полный пакет документов Важно
Финансы Минимальный заказ (MOQ) От 50 кг на позицию (фасовка 5–25 кг) Важно
Финансы Ценообразование Зависит от сорта и калибра. Скидки за объём и постоянство Важно
Финансы Условия оплаты Предоплата / по факту / отсрочка (для проверенных партнёров) Важно
Логистика Температура хранения Не выше +20 °C, влажность 20–50% Критично
Логистика Транспортировка Рефрижераторы или крытый транспорт Важно
Логистика Доставка по Казахстану Местные производители в Алматы — оперативная доставка по регионам Информация
💡 Что видно: 3 критичных пункта — декларация соответствия, качество сырья и температурный режим. Если поставщик не даёт документы или хранит при +25 °C — это сразу отказ.
